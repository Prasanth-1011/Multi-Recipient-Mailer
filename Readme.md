# Multi-Recipient Mailer

A Sleek, Professional, And Efficient Web Application Designed For Batch Email Distribution. Built With A Modern Tech Stack, It Allows Users To Send Personalized Or Bulk Messages To Multiple Recipients Seamlessly By Uploading Recipient Lists.

![App Screenshot](https://raw.githubusercontent.com/Prasanth-1011/Multi-Recipient-Mailer/main/Frontend/public/screenshot.png) _(Placeholder For Actual Screenshot)_

## Key Features

- **Batch Mailing**: Send Emails To Hundreds Of Recipients In A Single Operation.
- **Excel/CSV Support**: Effortlessly Parse Recipient Lists From Spreadsheet Files.
- **Modern Dark UI**: A Premium, Glassmorphic Interface Built With Tailwind CSS For An Enhanced User Experience.
- **Secure Communication**: Uses Nodemailer With Secure Transport Protocols.
- **Delivery History**: Track Sent Messages And Recipient Counts Via Integrated MongoDB Storage.

## Tech Stack

### Frontend

- **React 19**: Modern UI Library For Building Interactive Components.
- **Vite**: Ultra-Fast Build Tool And Development Server.
- **Tailwind CSS**: Utility-First CSS Framework For Custom Styling.
- **Lucide React**: Clean And Consistent Icon Library.
- **Axios**: Robust HTTP Client For API Interactions.

### Backend

- **Node.js & Express**: High-Performance Server-Side Environment And Framework.
- **MongoDB & Mongoose**: Flexible NoSQL Database For Storing Mail History.
- **Nodemailer**: Comprehensive Library For Sending Emails Via SMTP.
- **Bcrypt**: Secure Credential Hashing (For Admin/Security Features).
